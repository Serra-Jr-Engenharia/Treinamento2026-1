# 🚀 Programa de Trainee - Serra Jr. Engenharia

Olá, pessoal! Sejam muito bem-vindos e bem-vindas ao nosso programa de trainee!

Este repositório será o nosso ponto central de referência, onde vocês encontrarão o cronograma, materiais de apoio e links importantes.

O objetivo deste programa é fornecer uma base sólida em desenvolvimento de software, capacitando vocês a contribuírem em projetos reais e a crescerem profissionalmente dentro da nossa empresa júnior.

Preparem-se para muitos desafios, aprendizados e, claro, muita colaboração!

## 🎯 Cronograma do Programa

O programa é dividido em duas trilhas de conhecimento: Front-end e Back-end. As duas trilhas possuem a duração de 7 semanas, culminando em um projeto final para aplicar todo o conhecimento adquirido.

### 🎨 Trilha Front-end

- Semana 1 - Fundamentos de HTML e CSS

- Semana 2 - JavaScript Básico + DOM

- Semana 3 - JavaScript Avançado + APIs

- Semana 4 - Introdução ao React + Tailwindcss

- Semana 5 - Evoluindo no React + Consumo de APIs

- Semana 6 - React Router e Navegação

- Semana 7 - React com TypeScript

### ⚙️ Trilha Back-end

- Semana 1 - Fundamentos de Banco de Dados (MySQL)

- Semana 2 - Introdução ao Node.js, TypeScript e Fastify

- Semana 3 - Prisma e Conexão com MySQL

- Semana 4 - Validação com Zod

- Semana 5 - Autenticação e Autorização com JWT e Bcrypt

- Semana 6 - Relacionamentos no Prisma e Avançando no Backend

- Semana 7 - Deploy e Containers
